nid# 实现计划: WGSL Formatter VSCode Extension

## 概述

本实现计划将 WGSL Formatter VSCode Extension 分解为一系列增量式的编码任务。该扩展为 WebGPU Shading Language (WGSL) 提供代码格式化功能，采用分层架构设计，使用 TypeScript + VSCode API + esbuild + Vitest + fast-check 技术栈。

实现顺序遵循从核心到外围的原则：首先建立项目基础设施，然后实现核心格式化引擎，接着集成 VSCode API，最后添加配置和错误处理功能。
