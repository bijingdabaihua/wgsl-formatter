### 最简单的方法（3 步）

1. **打开终端**（在项目目录）

2. **运行安装命令**：
   ```bash
   code --install-extension wgsl-formatter-0.1.0.vsix
   ```

3. **打开测试文件并格式化**：
   - 在 VSCode 中打开 `test-example.wgsl`
   - 按 `Shift+Alt+F` (Windows/Linux) 或 `Shift+Option+F` (macOS)
   - 查看格式化效果！

### 使用自动化脚本

**Windows:**
```powershell
.\install-local.ps1
```

**Linux/macOS:**
```bash
chmod +x install-local.sh
./install-local.sh
```

## 🐛 常见问题

### Q: 安装后找不到插件？
**A:** 
1. 检查扩展面板 (`Ctrl+Shift+X`)
2. 搜索 "WGSL Formatter"
3. 确保插件已启用（不是禁用状态）

### Q: 格式化没有效果？
**A:**
1. 确保文件扩展名是 `.wgsl`
2. 检查输出面板是否有错误
3. 尝试重启 VSCode

### Q: 如何卸载？
**A:**
```bash
code --uninstall-extension wgsl-formatter.wgsl-formatter
```

### Q: 修改代码后如何重新测试？
**A:**
```bash
# 重新构建和安装
npm run package
.\install-local.ps1  # 或 ./install-local.sh
```
